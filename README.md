# Profit Mix Optimizer (Streamlit)

אפליקציית Streamlit שמציעה 3 חלופות לשילוב בין 2–3 קרנות השתלמות בהתאם ליעדים, מגבלות, ומשקל משמעותי גם לשירות.

## הרצה מקומית
```bash
pip install -r requirements.txt
streamlit run app.py
```

## קבצים
- `app.py` – האפליקציה
- `requirements.txt` – תלותים
- `קרנות_השתלמות_חשיפות.xlsx` – נתוני מקור (אפשר להחליף דרך UI)
